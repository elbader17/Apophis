Arsenal Técnico

¿Por qué APOPHIS redefine las auditorías?

Unifica el poder de un escáner autónomo de vulnerabilidades y un agente investigador de inteligencia en amenazas en un único servidor MCP.

Agentes de Caos Paralelos

Lanza 6 estrategias de auditoría concurrentes (`recon`, `aggressive`, `stealth`, `web`, `net`, `auth`) que compiten entre sí para encontrar fallas en el objetivo de forma eficiente.

Investigación en Vivo (Rss/APIs)

Sincroniza de forma dinámica las últimas vulnerabilidades de fuentes autoritativas como NVD NIST, CISA KEV, OSV, GHSA, Exploit-DB y feeds RSS especializados.

Generador de Stubs en Go

Compila stubs funcionales en Go con lógica preestructurada para comprobar vulnerabilidades específicas e intégralas como firmas duraderas dentro de la herramienta.

Inspección Web y SSL Profunda

Analiza debilidades SSL, cabeceras de seguridad ausentes, brute force de rutas sensibles expuestas (`.git/`, `.env`) y validaciones básicas de LFI, SQLi y XSS.

Totalmente Autocontenido

Compilado en un único binario estático en Go sin dependencias adicionales de librerías en caliente o pesados frameworks de virtualización externos.

Reportes en Markdown e Indexados

Cada auditoría se almacena tanto en formato JSON como Markdown estruturado. Perfecto para la indexación automática y fácil consumo de tu cliente IA.

API de Protocolo MCP

                
Herramientas expuestas para tu IA

                
Al conectar APOPHIS a tu cliente MCP de inteligencia artificial, tu LLM adquiere acceso directo a este set de capacidades de auditoría.

            


            
            

                
                
            


            
            
            

                

                    
                    

                        

                            apophis_audit
                            Completo
                        

                        
Escaneo concurrente multiruta sobre un host objetivo. Ejecuta validaciones de puertos, SSL/TLS, web app y credenciales por defecto.

                        

                            Args: { "target": "10.0.0.1", "strategy": "aggressive" }
                        

                    


                    
                    

                        

                            apophis_portscan
                            Rápido
                        

                        
Escaneo rápido de puertos TCP y recolección inteligente de banners de servicios (SSH, HTTP, FTP, etc.).

                        

                            Args: { "target": "domain.com", "top_ports": 100 }
                        

                    


                    
                    

                        

                            apophis_web_audit
                            Web OSINT
                        

                        
Auditoría web dirigida: valida políticas de cabeceras de seguridad, rutas críticas de depuración expuestas y pruebas sencillas de XSS/SQLi.

                        

                            Args: { "target": "https://api.test", "brute_paths": true }
                        

                    


                    
                    

                        

                            apophis_check_cve
                            Database
                        

                        
Cruce de firmas locales e información recolectada de versiones contra la base de datos de amenazas dinámica.

                        

                            Args: { "service": "apache", "version": "2.4.49" }
                        

                    

                

            


            
            

                

                    
                    

                        

                            apophis_research
                            Sync
                        

                        
Sincroniza y normaliza vulnerabilidades desde bases de datos globales de amenazas de ciberseguridad, actualizando el repositorio dinámico local.

                        

                            Args: { "sources": ["cisa-kev", "nvd"], "days_back": 15 }
                        

                    


                    
                    

                        

                            apophis_search_cve
                            Búsqueda
                        

                        
Filtra la base de datos sincronizada localmente por palabra clave, criticidad, puntuación CVSS mínima o filtros de CISA KEV.

                        

                            Args: { "keyword": "fortinet", "min_cvss": 8.0 }
                        

                    


                    
                    

                        

                            apophis_generate_stub
                            Generator
                        

                        
Construye stubs modulares en Go listos para la validación interna permanente de nuevas firmas del motor de caos.

                        

                            Args: { "cve_id": "CVE-2026-0001" }
                        

                    


                    
                    

                        

                            apophis_recommend_exploitation
                            Exploits
                        

                        
Genera comandos informativos y guías de explotación/remediación detalladas ante un CVE o hallazgo detectado.

                        

                            Args: { "report_id": "rpt-928d3a", "severity": "HIGH" }

Instalación y Configuración

                
Despliega APOPHIS en 3 Minutos

                
APOPHIS se distribuye como un binario estático autocontenido listo para enlazarse con hosts MCP locales.

            


            
            

                
                

                    
                    
                


                
                
                

                    
Clona el repositorio oficial desde GitHub y realiza la compilación estática usando la toolchain de Go:

                    
                    

                        
                        
# Clona el repositorio del proyecto
git clone https://github.com/elbader17/Apophis.git apophis
cd apophis

# Compila el servidor MCP principal
go build -o bin/apophis ./cmd/apophis

# (Opcional) Compila el target de pruebas de red local
go build -o bin/testtarget ./cmd/testtarget

                    

                    
Requisito: Disponer de Go v1.21 o superior instalado en el entorno de desarrollo local.

                


                
                

                    
Añade APOPHIS a la lista de servidores MCP en el archivo de configuración del host (por ejemplo, en ~/.config/opencode/opencode.json):

                    
                    

                        
                        
{
  "mcp": {
    "apophis": {
      "type": "local",
      "command": ["/RUTA/ABSOLUTA/HACIA/apophis/bin/apophis"],
      "enabled": true,
      "env": {
        "APOPHIS_STORE": "/home/TU_USUARIO/.apophis/reports",
        "APOPHIS_WORKERS": "6",
        "APOPHIS_TIMEOUT": "5s",
        "APOPHIS_NVD_KEY": "tu-nvd-api-key-aqui"  // Opcional
      }
    }
  }
}
